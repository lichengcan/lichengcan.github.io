    # "2024-03-24T02:02:57.000+00:00"转换------年月日 时分秒

    @JsonFormat(pattern = "yyyy-MM-dd HH:mm:ss", timezone = "GMT+8")

    全局事件格式化： 格式化全局时间字段 spring.jackson.date-format=yyyy-MM-dd HH:mm:ss # 指定时间区域类型 spring.jackson.time-zone=GMT+8

    这个时间字符串的格式是 ISO 8601 格式。ISO 8601 是国际标准化组织 (ISO) 发布的日期和时间表示法的标准。它以 "T" 字符分隔日期和时间部分，并且包含时区信息。

    具体来说，时间字符串"2024-03-24T03:03:39.000+00:00"的格式如下：

    • "2024-03-24" 表示日期部分，即年-月-日；
    • "T" 表示日期和时间的分隔符；
    • "03:03:39.000" 表示时间部分，即时:分:秒.毫秒；
    • "+00:00" 表示时区信息，其中 "+00:00" 表示 UTC 时区，也称为格林尼治标准时间 (GMT)。

    因此，这个时间字符串表示的是 "2024年3月24日，UTC时间下的03时03分39秒"。
