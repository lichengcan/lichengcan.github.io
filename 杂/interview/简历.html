    # 项目经验

    # 碳普惠合作项目:碳排放管理系统

    • 项目概述：低碳发展是可持续发展的必然途径，我国高度重视温室气体排放工作，鼓励个人和小微企业的低碳行为，将有利于促进低碳行为的全民参与性，从而促进绿色消费，进而推动企业低碳转型升级，提高整个社会低碳发展的水平，最终推动低碳经济、生态文明。
    • 技术框架: SpringBoot、SpringCloud、Nacos、Feign、MySQL、Mybatis-Plus、ACTable、Kafka等
    • 责任 & 亮点:
      • 参与了需求沟通、数据库设计、接口设计、接口实现、测试和文档编写。
      • 负责核算边界、碳排放因子、排放源、碳计算模型、活动数据填报模板、采集任务等模块的开发。
      • 使用 Sharding 进行垂直分表，优化了单表中部分字段较大、影响查询效率的问题。
      • 引入并发多线程解决碳计算速度较慢的问题，大幅度提高了用户的体验感。
      • 使用 A.C.Table 框架实现了自动建表功能。
      • 封装了多个常用工具类，包括雪花算法生成id、AOP切面防抖、控制层统一返回规范等.

    # 领导巡检服务平台

    • 项目概述：为了支持领导更直观地了解全县的发展状况，做出更科学、高效的决策，本项目汇总了县级各类数据进行分类展示。
    • 技术框架: Spring MVC、SpringBoot、Mybatis、MySQL、人大金仓数据库、druid数据源、Redis、Nginx等
    • 责任 & 亮点:
      • 完成了后台管理系统、登录、注册、权限控制、监测预警等模块的开发。
      • 成功对接了第三方海康开发能力平台。
      • 对原有的遗留的问题优化、优化了单表百万级的结构，采用水平分表和索引策略，针对车辆牌照唯一字段进行了优化。
      • 通过使用 Redis 缓存，将全县视频点位数据的查询时间从 10 秒+ 优化至约 2 秒。
      • 使用 Kafka 实现了首页事件广播功能。

    # 人寿后台管理系统平台

    • 项目概述：面对公司各部门间信息不同步和共享延误的问题，为促进部门间更高效、科学的协作与分工，以及简化管理流程、优化运营效率，我们拟建立一套“人寿后台管理系统平台”。该平台旨在全面解决公司信息服务的痛点，实现信息资源的即时共享，从而加强公司的整体运作协同。
    • 技术框架: SpringBoot、SpringCloud、shiro、MySQL、Mybatis-Plus、Nacos、Feign等
    • 责任 & 亮点:
      • 负责用户管理、角色管理、菜单管理、部门管理、岗位管理模块的开发。
      • 负责设计和实现文件加密传输工具类，涵盖了 AES 和 RSA 加密算法，保障了文件在互联网上传输过程中的安全性。
      • 开发了 XML 数据统计分析工具，提升了数据处理的效率和准确性。
      • 成功实现了文件加密传输工具类，确保了敏感数据在传输时的保密性，为数据安全提供了有力的保障。
